Проект базы данных для CRM контакт-центра.

Главная таблица [CRM].[Contacts], в ней информация: кто, кому, когда звонил и с каким результатом звонок закончился.

Для клиентов отдельная таблица [PersonalInfo].[Clients] с необходимыми данными, включая регион, который вынесен в таблицу [Dimensions].[Regions].

Для операторов создана таблица [CRM].[Operators] в которой хранится его данные как сотрудника, в какой он группе и под какой ролью - все 3 значения разнесены по таблицам:
	1.[PersonalInfo].[Employees]
	2.[OrgStructure].[Groups]
	3.[CRM].[Role]

Так же есть таблица содержащая информацию по депертаментам [OrgStructure].[Departments] и справочник результатов контакта [CRM].[ContactResults].
